# Java Glossary for Beginners

A cheat sheet of common terms to help you speak "Java".

| Term | Analogy | Definition |
| :--- | :--- | :--- |
| **Class** | Cookie Cutter | A blueprint that defines the structure and behavior of objects. |
| **Object** | The Cookie | A specific instance created from a Class blueprint. |
| **Variable** | Labeled Box | A container that stores data values. |
| **Method** | Action Button | A block of code that runs only when it is called (like a function). |
| **Parameter** | Input Slot | Information passed to a method so it can do its job. |
| **Return Type** | Output Type | The type of data (e.g., `int`, `String`) that a method gives back. |
| **Compiler** | Translator | A tool (`javac`) that turns your human-readable Java code into bytecode the computer understands. |
| **JVM** | Engine | The Java Virtual Machine that runs your compiled code on any computer. |
| **Primitive** | Atom | Basic data types built into Java (`int`, `double`, `boolean`, `char`). |
| **String** | Text Chain | A sequence of characters (it's an Object, not a primitive!). |
| **Public/Private**| Privacy Glass | Access modifiers that decide who can see or use your code variables/methods. |
| **Loop** | Repeat Cycle | Code that executes repeatedly based on a condition. |
